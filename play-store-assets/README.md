# Play Store icon assets

| File | Use |
|------|-----|
| `ic_launcher_512.png` | Upload as the **App icon** in Google Play Console (512×512) |
| `ic_launcher_source.png` | Original high-resolution source art |

Adaptive launcher icons scale the artwork inward so the heart and thumbs-up stay visible on round home-screen icons. The pink/lavender gradient lives on the background layer.

Launcher PNGs live under `app/src/main/res/mipmap-*`.
